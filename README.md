# ashare-research
A股多因子量化研究体系 | A-Share Multi-Factor Quant Research Framework

基于中证800成分股，覆盖因子挖掘、有效性检验、策略回测与绩效归因全链条的量化研究项目

研究样本 / Research Universe

股票池 / Universe: 中证800 (CSI 800, code: 000906)
回溯区间 / Backtest period: 2018-01-01 — 2024-12-31
数据源 / Data source: Akshare (公开免费接口 / Free public API)
基准 / Benchmark: 中证800全收益指数 / CSI 800 Total Return Index

项目核心研究目标
1. 构建覆盖基本面、技术面、另类维度的A股研究级核心因子库，完成因子的规范化预处理与全维度量化有效性检验
2. 基于有效因子构建多因子选股策略模型，通过历史回测开展收益、风险、换手维度的实证研究与稳健性验证
3. 沉淀标准化的量化研究分析框架与代码工具，形成符合部门研究规范的完整研究报告与实证成果
